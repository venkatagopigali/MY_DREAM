Gain complete mastery over Python syntax, structured programming, OOP principles, and data manipulation libraries.

  • Days 1-10: Syntax & Data Structures: Variables, Data Types (Int, Float, Complex, Boolean, String), String methods, List & List Methods, Tuple & Tuple Methods, Packing & Unpacking, Set & Set Operations, Dictionary & Dictionary Methods, List and Dict Comprehension.
  • Days 11-24: Control Flow & Function Design: Conditional statements (If, If-Else, If-Elif-Else, Nested If), Looping statements (For, While), Controlling statements (Break, Continue, Pass), Functions introduction, Types of parameters, Recursion with examples & scopes, Iterators & Generators, Callbacks.
  • Days 25-32: Object-Oriented Programming (OOP): Class, Object, Constructors (Default, Parameterized), Variables (Local, Global, Class/Instance variables), self keyword, Inheritance (Single-level, Multi-level, Multiple, Hierarchical, Hybrid), Polymorphism (Duck typing, Method overloading/overriding, Operator overloading), Encapsulation (Access modifiers: Public, Private, Protected, Name mangling), Abstraction.
  • Days 33-41: Exception Handling & Integrations: Exception Handling (Try, Except, Finally, Else), PDBC (Python Database Connectivity for SQL roles), PDBC projects, File Handling.
  • Days 42-62: NumPy & Pandas Data Libraries: NumPy array creation & properties, Indexing and Slicing, NumPy functions. Pandas DataFrames & Series, Inspection methods, Cleaning methods, Accessing methods, Read/Write & Joins, String functions, Group By, Filter, Query and conditions without Query, Pandas Cleaning Projects, Matplotlib visualizations, Capstone Project with Pandas & Matplotlib.

Learn relational database design, query optimization, joins, aggregation, and database administration tools.

  • Days 11-20: Relational Operations & Functions: Keys (Primary Key, Foreign Key), Joins (Inner, Left, Right, Outer, Cross, Self), Like, Between, As, Any, All, Set Operators (Union, Union All, Intersect, Except), Subqueries, Date Functions, String Functions, Window Functions, Views.
  • Days 21-31: Advanced Database Objects & Projects: Procedures, Triggers, TCL-DCL controls, Case, CTE (Common Table Expressions), Normalization rules, Indexes, Data Cleaning Project, SQL Final Project.

Master Excel spreadsheets, dynamic formulas, dashboards, lookups, and data structures.

  • Days 8-15: BI Tools & Data Modeling: Pivot Tables, Charts, DAX in Excel, Conditional Formatting, Data Modeling, Lookups (Vlookup, Hlookup, Xlookup), Index/Match formulas, Excel Capstone Project.

Master corporate data preparation, modeling, and automated report building.

  • Excel (Days 1-15): Ingesting files, math/text/logical/date formulas, cell references, conditional statements, Pivot Tables, Excel DAX, lookups (VLOOKUP, HLOOKUP, XLOOKUP), Index/Match, Capstone Project.
  • Power BI (Days 1-24): Ingestion, Power Query ETL, custom columns, append/merge, DAX aggregate/text/logical/date/filter/time-intelligence metrics, relational models (Star, Snowflake, Bus, Ring schemas), graphs, dashboard creation, cloud publishing.

Establish strong analytical foundations covering descriptive models, probability rules, distributions, hypothesis tests, and regressions.

  • Days 1-10: Probability & Descriptive Models: Introduction to statistics, types of data, population vs samples, mean, median, mode, variance, standard deviation, plotting (histogram, boxplot, scatterplot), probability basics, sample spaces, probability rules, conditional probability, Bayes Theorem, random variables, normal & binomial distributions, central limit theorem, sampling techniques & distributions.
  • Days 11-20: Inferential Statistics & Hypothesis Testing: Confidence intervals, hypothesis testing basics (null, alternative, p-value), t-test, z-test, chi-square test, ANOVA, Pearson & Spearman correlation, covariance, simple linear regression, multiple linear regression, regression evaluation metrics (R², adjusted R², MSE, RMSE).
  • Days 21-26: Regressions, ML Foundations & Time Series: Regression assumptions, logistic regression, probability in ML, bias-variance tradeoff, overfitting and underfitting, feature selection, PCA dimensions reduction, time series basics (trends, seasonality).

Build supervised classification and regression pipelines, and deploy unsupervised clustering models connected to cloud databases.

  • Days 1-11: Supervised ML & App Interface: Introduction to ML, ML workflow, classification vs regression, linear regression, multiple linear, logistic regression, Decision Trees, Random Forests, KNN, SVM, Streamlit dashboard deployment.
  • Days 12-23: Unsupervised ML & Database Serving: Unsupervised learning, K-Means clustering (concepts & coding), DBSCAN (concepts & coding), Hierarchical clustering, PCA, Apriori Association Algorithm, Aiven MySQL database connection, Streamlit web app.

Parse, tokenize, embed, and categorize text strings using state-of-the-art computational linguistics libraries.

  • Days 1-3: Text Representation: Preprocessing (tokenization, stop words, stemming, lemmatization, regex), Bag of Words, TF-IDF, N-Grams, NLP libraries (NLTK, spaCy), EDA on text data.
  • Days 4-6: Word Embeddings & Modeling: Word2Vec, GloVe, cosine similarity, text classification (Naive Bayes, Logistic Regression), evaluation metrics, Named Entity Recognition (NER), POS tagging, topic modeling, sentiment analysis.

Orchestrate Artificial, Convolutional, and Recurrent Neural Networks to solve computer vision and sequential tasks.

  • Days 1-3: ANN Architecture & Training: Introduction to Deep Learning, Artificial Neural Networks (ANN), training of neural networks (loss functions, backpropagation, and optimization algorithms).
  • Days 4-6: CNNs & Image Classification: Convolutional Neural Networks (CNN), image classification models using CNN layers.
  • Days 7-9: RNNs, Tooling & Serve: Recurrent Neural Networks (RNN), model evaluation techniques, deep learning tools (TensorFlow, Keras, PyTorch), deep learning mini project.

Intensive specialized weekend classes focusing on graphical statistics and exploratory scripting.

Master SQL and database design — essential for storing, querying, and managing application data in full stack projects.

Build responsive, dynamic, and beautiful web interfaces that connect seamlessly with your Python backends.

  • Days 1-12: HTML5 Foundations: HTML structure, Semantic tags (header, nav, section, footer, article), Forms & Input types, Tables, Media elements (img, video, audio), Hyperlinks, Metadata & SEO basics, HTML5 APIs.
  • Days 13-26: CSS3 & Responsive Design: Selectors & Specificity, Box Model, Flexbox, CSS Grid, Media Queries & Responsive layouts, Transitions & Animations, Pseudo-classes & Pseudo-elements, CSS Variables, Bootstrap 5 framework basics.
  • Days 27-40: JavaScript Essentials: Variables (var, let, const), Data Types, Operators, Conditions & Loops, Functions (regular, arrow, async/await), DOM Manipulation, Events & Event Listeners, Fetch API & AJAX, JSON handling, ES6+ features (destructuring, spread, modules), Mini project: Dynamic frontend connected to REST API.

Master Django — Python's most powerful web framework — to build secure, scalable, production-ready backend applications.

  • Days 1-12: Django Core Concepts: Django installation & project setup, MVT architecture (Model-View-Template), URL routing & path configurations, Views (Function-Based & Class-Based), Templates & Template language (tags, filters, inheritance), Static files management.
  • Days 13-25: Django ORM & Database Integration: Django ORM models, Field types & relationships (OneToOne, ForeignKey, ManyToMany), Migrations, QuerySets & filtering, Model Admin customization, Django Shell, Integration with MySQL & PostgreSQL databases.
  • Days 26-35: Forms, Authentication & Security: Django Forms & ModelForms, Form validation, User registration & login system, Django Authentication (login, logout, permissions, groups), Password hashing, CSRF protection, Session & Cookie management.
  • Days 36-45: Django REST Framework (DRF) & APIs: Introduction to REST APIs, DRF Serializers, APIView & ViewSets, Routers, Token Authentication & JWT, Filtering & Pagination, API testing with Postman, Capstone: Full CRUD REST API for a real-world app.

Build lightweight, fast microservices and REST APIs using Flask — perfect for AI-powered and microservice applications.

  • Days 1-10: Flask Fundamentals: Flask setup & application factory, Routes & HTTP methods (GET, POST, PUT, DELETE), Request & Response objects, Jinja2 templates, Static assets, Flask-SQLAlchemy ORM, Flask-WTF forms, Blueprint modular structure.
  • Days 11-20: Flask APIs & Deployment: Building RESTful APIs with Flask, Flask-RESTful extension, JSON responses, Error handling & HTTP status codes, JWT authentication with Flask-JWT-Extended, Connecting Flask to MySQL/PostgreSQL, API testing, Capstone: AI model serving via Flask REST API (deploying an ML prediction endpoint).

Deploy real full stack applications to the cloud and build a job-ready portfolio with 3 complete projects.

  • Days 15-20: Capstone Full Stack Projects:
    • Project 1: Job Portal Web App (Django + MySQL + Bootstrap — full CRUD with authentication)
    • Project 2: E-Commerce Product Catalog (Flask REST API + JavaScript frontend + PostgreSQL)
    • Project 3: AI-Powered Chatbot App (Django + OpenAI API + DRF — deployed to cloud)
